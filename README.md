# DS266 Final Project - Wes Morberg
## Title: Detection and Analysis of Monetary Policy Signals in Central Banks Minutes with TF-IDF, FinBERT, and ModernBERT
---
### Contents:
- 'data_pipeline.ipynb': scrape central bank minutes text and assign binary rate change labels for modeling
- 'modeling_and_visualization.ipynb': runs all four model configs and generates Figure 1 in report
- 'linguistic_analysis.ipyn': TF-IDF feature coefficient check, TTR analysis, phrase frequency analysis in ModernBERT false negative minutes text
- 'central_bank_minutes_labeled.csv': final labeled dataset used in the report (696 total meeting minutes across FOMC, BoE, and RBA)
